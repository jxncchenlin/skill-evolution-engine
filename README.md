# Skill Evolution Engine

> AI Skill 的自我进化引擎 — 从第一性原理出发，持续感知、分析、改进所有已安装 Skill，并将改进模式跨 Skill 扩散。

## 核心架构

五层闭环架构 + 两层前置捕获：

```
Layer 0   实时捕获层（对话中非侵入式提取经验）
Layer 0.5 话题指纹+切换检测（避免重复检索）
Layer 0.6 经验主动提醒（Skill加载时推送历史经验）
Layer 1   感知层（信号收集与分类）
Layer 2   分析层（第一性原理5问链诊断）
Layer 3   改进层（VFM评分 + FP模式匹配 + 风险分级执行）
Layer 4   验证层（效果追踪 + FP置信度管理）
Layer 5   扩散层（跨Skill模式传播）
```

## 核心特性

- **第一性原理诊断**：5问链（假设→违反→影响→根因→修复）精准定位问题
- **VFM改进评分**：频率×3 + 影响面×3 + 自动可修复×2 + 复合杠杆×2 + 扩散价值×2
- **FP模式库**：已验证的改进模式，带置信度管理，驱动快速通道
- **效果追踪**：改进后下次使用时运行时验证（有效/部分有效/无效/有副作用）
- **经验分层存储**：GK（通用知识）+ SE（技能经验）双层结构
- **话题指纹**：避免同话题重复检索，切换时自动重载经验
- **风险分级执行**：Low/Medium自动执行，High需用户确认

## 设计理念

这个引擎的核心理念是：**让AI不仅会执行任务，还能从执行中学习，并将学习成果扩散到整个Skill生态。**

灵感来源于软件工程中的 CI/CD 流水线和生物学中的进化机制——感知变异、评估适应性、保留有益突变、传播优势基因。
